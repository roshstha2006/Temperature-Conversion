# Temperature Converter

A simple Python program that converts temperature:
- Celsius to Fahrenheit
- Fahrenheit to Celsius

## Features
- Takes user input from terminal
- Converts Celsius → Fahrenheit
- Converts Fahrenheit → Celsius
- Beginner-friendly Python code

## How It Works
1. User enters temperature in Celsius → program prints Fahrenheit.
2. User enters temperature in Fahrenheit → program prints Celsius.

## How to Run
1. Make sure Python is installed.
2. Open terminal in the project folder.
3. Run the program:

```bash
python temperature_converter.py
